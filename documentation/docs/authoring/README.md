# authoring

This directory contains guidelines on how to organize scholarly content in subfolders.


## Organization

``` yaml
authoring/
├── book-chapters
│   └── README.md
├── books
│   └── README.md
├── dissertation
│   ├── final_dissertation.pdf
│   ├── README.md
│   ├── references
│   └── source
├── manuscripts
│   ├── README.md
│   ├── references
│   ├── title-A
│   ├── title-B
│   └── title-C
├── papers
│   ├── 001-YYYY-Journal-Title
│   ├── 002-YYYY-Journal-Title
│   ├── 003-YYYY-Journal-Title
│   └── README.md
├── presentations
│   └── README.md
├── proposals
│   └── README.md
├── README.md
├── reports
│   └── README.md
├── teaching
│   └── README.md
├── thesis
│   ├── final_thesis.pdf
│   ├── README.md
│   ├── references
│   └── source
└── tutorials
    ├── HandsOnVASP
    ├── QuantumEspresso
    └── README.md
```

!!! warning

    This is only an indicative structure and can be modified based on the project requirements.
